# Bead 0002 - Ontology Registry

## Objective

Implement a minimal ontology registry that can resolve terms from a Turtle ontology file for early validation and testing.

## Tasks

- Implement a simple ontology registry backed by an ontology source.
- Provide tests for lookup by label and URI.
- Document limitations and planned upgrades.

## Acceptance Criteria

- Registry can resolve `Asset` and other core terms from `asset-ontology.ttl`.
- Tests pass without external dependencies.
- Future work notes captured in the BMAD spec.
