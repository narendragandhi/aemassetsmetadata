# Bead 0015 - Graph Storage Design

## Objective

Define the persistent graph storage adapter strategy and document migration path from in-memory storage.

## Tasks

- Define a graph storage adapter contract beyond the in-memory store.
- Document candidate backends and integration approaches.
- Add a configuration placeholder for selecting storage mode.

## Acceptance Criteria

- Architecture doc includes storage options and trade-offs.
- A configuration placeholder exists for future graph adapter selection.
